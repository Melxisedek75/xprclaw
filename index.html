FAQ

Common questions

Is this a real app or just a website?
GCSC CLAW is a Progressive Web App (PWA) — a modern web standard supported by Apple, Google, and Microsoft. Once installed, it behaves exactly like a native app: appears on your home screen, runs in its own window, works offline, and can send notifications. PWAs are used in production by Twitter, Spotify, Uber, Starbucks, and many others.
Why not the App Store or Google Play?
PWAs install directly from the web — no app store approval, no delays, no developer fees, no store commissions. Native iOS/Android versions are on our roadmap, but PWA offers the same experience today with faster updates and full transparency. You always have the latest version.
Is it safe to install?
Yes. PWAs run in your browser's security sandbox — they cannot access your OS, files, contacts, or other apps. We don't track you, we don't require an account, and all code is open-source on GitHub. You can review every line before installing. Uninstall anytime by removing from your home screen.
Does it work offline?
Yes. After first install, the entire app is cached locally. You can open it, browse all pages, review contracts, and use the AI Secretary even with no internet. Transactions on XPR Network require connectivity when you submit them.
Do I need a crypto wallet to use it?
No. You can explore all features (dashboard, chat, trading bot UI, marketplace) without a wallet. A wallet is only needed to actually stake $XLAW, deploy an agent on-chain, or purchase skills. Supported: WebAuth, Proton Wallet, Metal Pay.
What does the trading bot do?
The app includes a UI to monitor a Python day-trading bot that runs as a separate backend service. The bot uses a hybrid ML + LLM architecture (trend-following, mean-reversion, regime detection, news filtering). Source code is in the ZIP under backend/trading-bot/. Warning: algorithmic trading carries real financial risk. Shadow-trade for at least 2 weeks before running live.
What if my browser doesn't show an install button?
Not all browsers support PWA install prompts. If the button doesn't work: iOS — use Safari and the Share menu. Android — use Chrome or Brave and the ⋮ menu. Desktop — use Chrome, Edge, or Brave. If you still have trouble, download the HTML file and open it directly.
