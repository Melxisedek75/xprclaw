FAQ

Common questions

Is this a real app or just a website?
GCSC CLAW is a Progressive Web App (PWA) — a modern standard supported by Apple, Google, and Microsoft. Once installed, it behaves like a native app: home screen icon, standalone window, offline support, push notifications.
Why not the App Store or Google Play?
PWAs install directly from the web — no store approval, no delays, no commissions. Native iOS/Android versions are on the roadmap. You always get the latest version automatically.
Is it safe to install?
Yes. PWAs run in the browser's security sandbox. We don't track you, don't require an account, and all code is open-source on GitHub. Uninstall anytime by removing the icon.
Do I need a crypto wallet?
No. You can explore all features without a wallet. A wallet is only needed to stake $XLAW, deploy agents on-chain, or trade real funds. Supported: WebAuth, Proton Wallet, Metal Pay.
What does the trading bot do?
The bot uses a hybrid ML + LLM architecture — trend-following, mean-reversion, regime detection, and news filtering. Currently runs in paper trading mode (no real funds). Real trading mode requires manual activation. Algorithmic trading carries real financial risk.
What is $XLAW?
$XLAW is the utility token of GCSC CLAW on XPR Network. It's used to pay for AI agent execution, staking rewards, governance voting, and marketplace skill purchases. Contract: gcsctoken111.
